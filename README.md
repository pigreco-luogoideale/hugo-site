# Il sito di PiGreco

Quasi tutto nel sito è pubblico e viene messo su Github.

Si fa push sul repo, dove si può modificare il contenuto.

Per pubblicarlo, per ora è necessario fare pull, eseguire hugo e copiarlo sul
server FTP.
